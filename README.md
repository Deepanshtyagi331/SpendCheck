# SpendCheck: AI Spend Audit Tool

SpendCheck is a "Mint.com for AI subscriptions" that helps startup founders and engineering managers discover if they are overpaying for AI tools. Built for the Credex recruitment assignment, this tool provides instant audits and actionable saving recommendations.

## Features
- **Comprehensive Tool Support**: Audit Cursor, Copilot, Claude, ChatGPT, and more.
- **Instant Financial Insights**: Get monthly and annual savings calculations based on real pricing data.
- **Personalized AI Summaries**: Every audit includes a custom summary generated by Claude 3.5 Sonnet.
- **Shareable Reports**: Unique public URLs for every audit to facilitate sharing and viral growth.

## Quick Start

### Installation
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Deployment
Deploy to Vercel:
```bash
vercel
```

## Decisions
1. **Shadcn/UI over Tailwind Component Libraries**: Chose shadcn for its "copy-paste" nature, allowing deep customization of the design system to match Credex's premium aesthetic.
2. **Supabase for Persistence**: Used Supabase for its ease of use with Next.js and high-performance Postgres backend, ideal for rapid prototyping and reliable data storage.
3. **Hardcoded Audit Logic**: Opted for hardcoded finance-literate logic instead of AI for the core audit to ensure 100% accuracy and defensibility of financial recommendations.
4. **LocalStorage for Form State**: Implemented local persistence for the input form to ensure founders don't lose progress during the multi-tool audit process.
5. **Resend for Transactional Email**: Selected Resend for its developer-friendly API and high deliverability, crucial for the "save report" lead capture flow.
